## OpenVINO backend

Intel OpenVINO is a library which brings optimized deep learning primitives for IA.

## Config options

There is a config options related to OpenVINO backend:

* `use_openvino` - `True` to enable OpenVINO backend. `False` to use default networks backend (TensorFlow)
